TILKEE Connector for SugarCRM

This connector is design to access to TILKEE server information from your SugarCRM instance

see below for more information

ChangeLog

version 1.0

  • EAPM access to connect to tilkee
  • External API access to Tilkee REST service
  • Set and Access to projects Tilkee Stats
  • Links Tilkee brochure to Contacts, Accounts and Leads
  • Links Tilkee proposal to Opportunities

Installation

1 - Install the Tilkee Connector module with SugarCRM module loader

2 - Go to "Connector Settings / Enable Connectors" to activate connector for users and select modules available to access Tilkee information

3 - Go to "Tilkee / General parameters" to configure TILKEE server URL: https://app.tilkee.com

4 - Users have to create an external link in their profil to connect to their Tilkee account: go to https://app.tilkee.com/oauth/applications to creat a new application, with return URL "http://YOU_SUGAR_URL/retourToken.php"

5 - Retrive client id to update "TILKEE Client ID" in Tilkee / General parameters" (you don't secret key)

6 - Now a Tilkee subpanel appear in selected objects to access to project stats or to select a project (proposal or brochure)

Remark : sometimes it is necessary to "Rebuild and Repair" to clean the ExternalAPI cache file

FAQ

TILKEE doesn't appear in application list when I want to create an external link
In the Administration screen, go to Repair and "Quick Repair and Rebuild"
